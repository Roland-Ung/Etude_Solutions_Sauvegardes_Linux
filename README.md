# Etude comparatifs des solutions de sauvegardes sous Linux

## Objectifs
- Déployer et configurer plusieurs solutions de sauvegarde sous Linux
- Mettre en œuvre des sauvegardes distantes via SSH
- Tester les procédures de restauration
- Analyser les différences de performance
- Étudier les aspects de sécurité (chiffrement, confidentialité, ransomwares, intégrité)

## Environnement
- Debian / Ubuntu
- station1 (machine cliente)
- serveur2 (machine serveur)
